# Bmbp 快速开发平台

## 平台简介

开箱即用的信息化开发平台，适用于 CURD 数据采集系统建设，提供以应用为单元的数据采集、流程审批、单据打印、统计分析、大屏展示的五大类场景的信息化建设。
